# XML Tag Reference for System Integrity

The system requires specific XML tags to validate AI output. If these tags are missing or incomplete, the system will reject the output and retry.

1. **Storyteller Output (Story)**
   - Tag: `<game>...</game>`
   - Usage: Wraps the raw story text generated by AI-1.

2. **Director Output (Final)**
   - Tag: `<finally>...</finally>`
   - Usage: Wraps the final text with stage directions (e.g., [Background-X]) generated by AI-2.

3. **Small Summary**
   - Tag: `<summary_little>...</summary_little>`
   - Usage: Wraps the 2-3 sentence summary of recent conversation.

4. **Big Summary**
   - Tag: `<summary_big>...</summary_big>`
   - Usage: Wraps the cohesive global story summary.

5. **Plot Guidance (Planner)**
   - Tag: `<guide>...</guide>`
   - Usage: Wraps the JSON output or text advice from the Architect (AI-3).

**Note:** Ensure all content is strictly inside these tags. Any text outside will be discarded or cause validation failure.
